# 隐私政策 / Privacy Policy

**最后更新 / Last updated: 2026-07-09**

## 中文版

本隐私政策说明 **The Parliament Bot**（以下简称“本 Bot”）在 Discord 上运行时如何收集、
使用和存储数据。将本 Bot 添加到服务器，即表示服务器管理员及其成员同意本政策。

### 1. 我们收集哪些数据
本 Bot 仅为实现其功能而收集必要数据，包括：
- Discord 标识信息：用户 ID、身份组 ID、频道 ID、服务器 ID；
- 成员状态：成员是否在服务器中、身份组同步状态；
- 安全数据：邀请记录、黑名单；
- 管理数据：处罚（警告 / 禁言）记录；
- 活跃度数据：在被管理员设置为“监控”的频道内的发言条数、被提及 / 主动提及次数
  （仅统计数字，不含消息文本内容）；
- 功能配置：管理员设置的各项配置；
- 在使用总结 / 归档功能时涉及的消息内容（见第 3 节）。

本 Bot 不会收集你在 Discord 之外的任何个人信息。

### 2. 我们如何使用这些数据
数据仅用于实现本 Bot 的功能，包括：跨服务器身份组同步、受控邀请与防炸群、
自助身份组申请、风纪处罚、违禁词自动清理、频道内容总结与内容归档。

### 3. 关于消息内容
- 违禁关键词审核：消息内容仅在内存中即时匹配，**不做存储**。
- 频道 AI 总结：管理员主动执行总结时，所选时段的消息内容会被发送至第三方 AI 接口
  （兼容 OpenAI 的接口）以生成摘要，生成的摘要可能被保存。
- 内容归档 / 迁移：仅在管理员主动触发时，将指定内容导出保存至本 Bot 自己的服务器。

### 4. 数据存储与保留
- 数据存储在本 Bot 运行的服务器上的本地数据库（SQLite / JSON 文件）中；
- 数据仅保留到实现相应功能所需的期限；审计日志约保留 90 天后自动清理；
- 我们采取合理措施保护数据安全，但无法保证绝对安全。

### 5. 数据共享
除下列情况外，我们不会出售、出租或向任何第三方分享你的数据：
- 使用 AI 总结功能时，向所配置的第三方 AI 接口发送消息内容以生成摘要；
- 法律要求时。

### 6. 你的权利与选择
- 上述功能均由服务器管理员按频道配置和触发，你可通过所在服务器管理员调整配置来退出；
- 你可通过下方联系方式，要求查询或删除与你相关的数据。

### 7. 年龄要求
本 Bot 服务于可能包含成人（NSFW）内容的社区，仅面向 18 岁及以上、且达到你所在地区
法定成年年龄（以较高者为准）的用户。未满上述年龄者不得使用本 Bot 或相关服务。

### 8. 政策变更
本政策可能不时更新，更新后将修改顶部“最后更新”日期。

### 9. 联系方式
如有任何问题，请通过 **Discord** 前往我们服务器的 **意见与投诉投稿** 频道与我们联系。

## English Version

This Privacy Policy explains how **The Parliament Bot** ("the Bot") collects, uses, and
stores data while operating on Discord. By adding the Bot to a server, server
administrators and their members agree to this policy.

### 1. Data We Collect
The Bot collects only the data necessary to provide its features:
- Discord identifiers: user IDs, role IDs, channel IDs, server IDs;
- Membership state: whether a member is in a server, role-sync state;
- Security data: invite records, blacklist;
- Moderation data: punishment records (warnings / mutes);
- Activity data: message counts and mention counts in channels an admin has set to
  "monitored" (counts only, not message text);
- Feature configuration set by administrators;
- Message content when the summary/archival features are used (see Section 3).

The Bot does not collect any personal information from outside Discord.

### 2. How We Use Data
Data is used solely to provide the Bot's features: cross-server role synchronization,
controlled invites and anti-raid protection, self-service role requests, moderation,
automatic keyword removal, channel summaries, and content archival.

### 3. Message Content
- Keyword moderation: content is matched in memory only and is **not stored**.
- AI channel summaries: when an admin runs a summary, the messages in the selected
  range are sent to a third-party AI API (OpenAI-compatible) to generate a summary;
  the resulting summary may be stored.
- Archival/migration: only on explicit admin action, exported to the Bot's own host.

### 4. Storage and Retention
- Data is stored in a local database (SQLite / JSON files) on the host running the Bot;
- Data is retained only as long as needed for the relevant feature; audit logs are
  cleared automatically after about 90 days;
- We take reasonable measures to protect data but cannot guarantee absolute security.

### 5. Data Sharing
We do not sell, rent, or share your data with any third party, except:
- Sending message content to the configured third-party AI API when the AI summary
  feature is used;
- When required by law.

### 6. Your Rights and Choices
- All features above are configured and triggered per-channel by server
  administrators; you may opt out via your server's administrators;
- You may request access to or deletion of your data via the contact below.

### 7. Age Requirement
The Bot operates in communities that may contain adult (NSFW) content and is intended
only for users who are 18 or older and have reached the age of majority in their
jurisdiction, whichever is higher. Anyone below that age may not use the Bot or its
related services.

### 8. Changes
This policy may be updated from time to time; the "Last updated" date above will
change accordingly.

### 9. Contact
For any questions, contact us at **Disord** in our server channel 
**意见与投诉投稿**.
